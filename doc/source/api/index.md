# aas-core 3.1 C# API

The API of the SDK is split into six different units:

| Namespace                                                                | Description                                        |
|--------------------------------------------------------------------------|----------------------------------------------------|
| [AasCore.Aas3_1](AasCore.Aas3_1.yml)                           | Create the model programmatically                  |
| [AasCore.Aas3_1.Visitation](AasCore.Aas3_1.Visitation.yml)     | Iterate over and transform the model               |
| [AasCore.Aas3_1.Constants](AasCore.Aas3_1.Constants.yml)       | Specify constants such as constant sets of strings |
| [AasCore.Aas3_1.Verification](AasCore.Aas3_1.Verification.yml) | Verify that the model fulfills the constraints     |
| [AasCore.Aas3_1.Jsonization](AasCore.Aas3_1.Jsonization.yml)   | De/serialize from/to JSON                          |
| [AasCore.Aas3_1.Xmlization](AasCore.Aas3_1.Xmlization.yml)     | De/serialize from/to XML                           |
| [AasCore.Aas3_1.Enhancing](AasCore.Aas3_1.Enhancing.yml)       | Wrap and enhance instances of the model            |

Please refer to [Getting Started](../getting_started/intro.md) for a brief introduction on how to use the API.
